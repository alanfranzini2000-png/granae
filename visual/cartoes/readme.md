# Documentacao de `visual/cartoes`

Diretorio de agrupamento das artes de cartoes de viagem.

## Estrutura

- `cartões/`: contem os PNGs finais dos cartoes.

## Uso

Essas artes parecem ser fontes/versoes de design. O frontend atual usa imagens de viagem em `plataforma/frontend/public/viagens`, nao esta pasta diretamente.

## Cuidados

- Se um cartao virar asset runtime, copie para `plataforma/frontend/public/viagens` ou outro subdiretorio publico e atualize o codigo que referencia o arquivo.
