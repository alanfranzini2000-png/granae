## Uso recomendado

Use estes arquivos como material de origem para gerar/exportar variantes publicas em `plataforma/frontend/public/gogo`.

## Cuidados

- Os nomes nao sao ideais para paths de app por terem espacos e acentos. Para uso no frontend, exporte com nomes consistentes como `GG.familia.N.PNG`.
- Antes de apagar uma imagem daqui, confira se ela nao e a unica fonte de alguma variante ja publicada.
